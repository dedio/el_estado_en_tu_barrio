# El Estado en tu Barrio
_Aplicación para la administración del programa «El estado en tu barrio» del Gobierno de Ciudad de Buenos Aires

## Pre requisitos
* Python3.9
* Django 4.1.3
* Virtualenv

### Instalación
...
virtualenv venv
...
...
venv/bin/activate
...
...
pip install -r requirements.txt
...
